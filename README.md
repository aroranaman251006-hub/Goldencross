Core Functionality: An automated trading-analysis tool that processes, analyzes, and visualizes 5 years of historical OHLCV market data specifically for DBS Group (D05.SI).

Technical Infrastructure: Built with a custom Python ETL pipeline engineered to extract, clean, and resolve data-type mismatches before seamlessly integrating the clean data into Power BI for visual reporting.

Algorithmic Alerts: Generates real-time BUY/SELL signals derived from 50-day and 200-day Simple Moving Average (SMA) crossovers.

Risk Mitigation: Minimizes traditional lagging-indicator risk by pairing the moving averages with a 14-day RSI oscillator, which serves as a secondary confluence filter to verify overbought and oversold market conditions.

Timeline: Completed in December 2025.
